# Crop Yield Prediction using Machine Learning

This project predicts crop yield using agricultural and climate data using machine learning models.

## Project Structure

- dataset : Contains the dataset in csv format
- notebook : Google colab notebook with code
- Project report : Project documentation

## Models Used
- Linear Regression
- Random Forest Regressor

## Evaluation Metrics
- RMSE
- MAE
- R2 Score

## How to Run
1. Open the notebook in Google Colab or Jupyter
2. Upload the dataset
3. Run all cells
